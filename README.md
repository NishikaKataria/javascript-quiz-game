# JavaScript Quiz Game - Lab Assignment 4

This is my Lab 4 project. It is a simple quiz game that runs in the browser console.

## Features (as required in the assignment)
- Questions stored in an array
- For loop to ask each question
- prompt() for user input
- alert() for feedback
- Answers checked using toLowerCase() and trim()
- Score increases on correct answers
- Final score shown at the end
- runQuiz() function used for structure

## How to Run
1. Open index.html (or any webpage).
2. Right click → Inspect → Console.
3. Paste the code from quiz.js into the console.
4. Press Enter to start the quiz.
